# Session Storage Tests

## Folder Purpose
This folder contains test suites focused on validating session and local storage behavior in web applications using Playwright.

## Description of Contents
- `228_Session.spec.ts`: Tests for session storage management.
- `229.TestVWo.spec.ts`: Tests for storage validation on VWO.

## Usage Instructions
To run the tests in this folder:
```bash
npx playwright test tests/04_Session_Storage
```

## Notes
Ensure the target applications are accessible and the necessary session states are considered during test execution.
