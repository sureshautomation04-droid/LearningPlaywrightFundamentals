# Multiple Element Tests

## Folder Purpose
This folder contains test suites for handling collections of elements, iterating through multiple matching locators, and performing bulk actions.

## Description of Contents
- `231_Multiple_Element.spec.ts`: Tests for interacting with multiple similar elements on a page.

## Usage Instructions
To run the tests in this folder:
```bash
npx playwright test tests/06_Multiple_Element
```

## Notes
Focuses on the use of `locator.all()` and looping mechanisms to validate multiple elements.
